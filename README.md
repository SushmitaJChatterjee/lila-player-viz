# LILA BLACK — Player Journey Visualizer

A web-based visualization tool for Level Designers to explore 
player behavior in LILA BLACK.

## Live Tool
👉 https://lila-player-viz-emursidvzejvfnaddfdjxn.streamlit.app/

## Features
- Player paths on minimap
- Kill, death, loot, storm event markers
- Human vs bot visual distinction
- Filter by map, date, match
- Heatmaps for kill/death/traffic zones
- Match timeline

## How to run locally
1. Clone this repo
2. Install dependencies: pip install -r requirements.txt
3. Run: streamlit run app.py
